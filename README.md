Welcome to my personal portfolio project. This application is a high-performance web platform designed to showcase my technical skills, professional experience, and creative projects. Built with FastAPI for a modern, asynchronous backend and deployed on Vercel for seamless global access.

Key Features

Asynchronous Processing: Utilizes FastAPI’s speed for efficient request handling.

Responsive UI: A clean, mobile-first design that adjusts to any screen size.

Dynamic Routing: Seamless navigation between different sections of the portfolio.

SEO Optimized: Structured HTML and metadata to improve search engine visibility.

Cloud Deployment: Fully integrated with Vercel’s serverless architecture.

 Technology Stack

Backend: Python 3.x, FastAPI.

Frontend:  HTML5, CSS3, JavaScript.

Templating:  Jinja2 for dynamic content rendering.

Server:  Uvicorn (ASGI server implementation).

Deployment:  Vercel (using vercel.json configuration).

📁 Project Directory Structure

Based on the development environment, the project is organized as follows:

app.py:  The core application file containing API routes.

static/: Contains static assets like style.css, script.js, and profile.jpg.

templates/: Holds the index.html file and other UI components.

requirements.txt: Lists all Python packages required for the project.

vercel.json: Configuration for hosting on Vercel.

.env: Securely stores environment variables.

Local Setup and Installation
To run this project on your local machine, follow these steps:

•	Clone the repository.

•	Create a virtual environment: python -m venv .venv.

•	Install dependencies: pip install -r requirements.txt.

•	Run the server:  Run app.py

•	Open in your browser :
